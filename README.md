# Shifty Maze!

Play it at [yousafe.github.io/shifty-maze/](https://yousafe.github.io/shifty-maze/)!


### Project Setup

```sh
npm install
cd game-core
npm install
npm run build
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

And remember to `npm run build` the `game-core` when you make changes to it.
